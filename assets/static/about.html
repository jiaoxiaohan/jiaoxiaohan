This website displays Kineticharts, an animation design scheme for creating charts that express five positive affects (joy, amusement, surprise, tenderness, and excitement). Regarding each affect, we designed varied kinetic motions represented by bar charts, line charts, and pie charts, resulting in 60 animated charts for the five affects. In the “Home” tab, we present the 60 designed animations as 60 method cards. Each chard shows the name of the animation, a brief description, and a GIF demo. Users can hover on the cards to see kinetic effects. The color of each card encodes the category of the affects while the icons encode chart types and editorial layers.

Our website also provides the stimuli we used in a user study (Study II in the paper) as well as a gallery to demonstrate the possible application of Kineticharts in real-world data stories.
